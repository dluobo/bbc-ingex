Building ACE and TAO on Linux

Obtain source code

Get tar file and extract:

wget ftp://download.dre.vanderbilt.edu/ACE+TAO-distribution/ACE+TAO.tar.bz2
tar xf ACE+TAO.tar.bz2

Set environment variables

You will get a directory ACE_wrappers. Set environment variable ACE_ROOT to point to this directory, for example:

export ACE_ROOT=/usr/local/src/ACE_TAO_x52/ACE_wrappers

Set environment variable TAO_ROOT to point to ACE_wrappers/TAO, for example:

export TAO_ROOT=$ACE_ROOT/TAO

Set LD_LIBRARY_PATH to include $ACE_ROOT/lib, for example:

export LD_LIBRARY_PATH=$LD_LIBRARY_PATH:$ACE_ROOT/lib

Create config files

Create a file $ACE_ROOT/ace/config.h containing:

#include "ace/config-linux.h"

Create a file $ACE_ROOT/include/makeinclude/platform_macros.GNU containing:

include $(ACE_ROOT)/include/makeinclude/platform_linux.GNU

Note there is no # before include

Build

Now do some makes:

cd $ACE_ROOT/ace
make ACE

cd $ACE_ROOT/apps/gperf
make gperf

cd $TAO_ROOT
make Naming_Service

The latter will make all you need for standard CORBA applications.

The -j2 argument can be used with make to run two jobs concurrently, which may speed things up.
